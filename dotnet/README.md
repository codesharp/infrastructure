## NET40

* Build

```shell
build.bat nuget
build.bat all [pack]
build.bat test_core
build.bat test_framework
```

## MONO

just use dll under mono 2.10.8+

or build:
```shell
sh build.sh nuget
```
or
```shell
sh nuget.install.sh
```
then
```shell
sh build.sh all [pack]
sh build.sh test_core
sh build.sh test_framework
```
